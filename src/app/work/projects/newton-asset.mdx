{/* BAGIAN 1: CHALLENGE */}
<En>
## The Challenge
Managing hundreds of IT assets (laptops, printers, servers) manually using spreadsheets is prone to errors. The IT department struggled to track asset movement, warranty status, and maintenance history, leading to lost equipment and unoptimized budget planning.
</En>

<Id>
## Tantangan
Mengelola ratusan aset IT (laptop, printer, server) secara manual menggunakan spreadsheet sangat rentan terhadap kesalahan. Departemen IT kesulitan melacak pergerakan aset, status garansi, dan riwayat pemeliharaan, yang menyebabkan hilangnya peralatan dan perencanaan anggaran yang tidak optimal.
</Id>


{/* BAGIAN 2: SOLUTION */}
<En>
## The Solution
I built **Newton Asset Management**, a centralized web-based inventory system. It digitizes the entire asset lifecycle from procurement to disposal. The system features a secure login portal, barcode scanning for quick identification, and automated alerts for warranty expirations.
</En>

<Id>
## Solusi
Saya membangun **Manajemen Aset Newton**, sistem inventaris berbasis web yang terpusat. Sistem ini mendigitalkan seluruh siklus hidup aset mulai dari pengadaan hingga pemusnahan. Fitur utamanya meliputi portal login yang aman, pemindaian barcode untuk identifikasi cepat, dan peringatan otomatis untuk masa garansi yang akan habis.
</Id>


{/* BAGIAN 3: FEATURES */}
<En>
## Key Features
- **Lifecycle Tracking:** Monitor asset status (Active, In-Repair, Disposed) in real-time.
- **Check-in/Check-out:** Record who is holding which device with a digital handover log.
- **Maintenance Scheduler:** Automated reminders for regular hardware maintenance.
- **Depreciation Calculator:** Automatically calculates asset value reduction over time for finance reporting.
</En>

<Id>
## Fitur Utama
- **Pelacakan Siklus Hidup:** Pantau status aset (Aktif, Sedang Diperbaiki, Dimusnahkan) secara real-time.
- **Check-in/Check-out:** Catat siapa yang memegang perangkat apa dengan log serah terima digital.
- **Penjadwal Pemeliharaan:** Pengingat otomatis untuk perawatan perangkat keras rutin.
- **Kalkulator Depresiasi:** Secara otomatis menghitung pengurangan nilai aset dari waktu ke waktu untuk pelaporan keuangan.
</Id>


{/* BAGIAN 4: TECH */}
<En>
## Technologies Used
- **Laravel:** Backend framework for secure data handling and authentication.
- **MySQL:** Relational database to store thousands of asset records efficiently.
- **Bootstrap 5:** Clean and responsive user interface for admin dashboard.
- **DataTables:** For advanced searching, sorting, and filtering of asset lists.
</En>

<Id>
## Teknologi yang Digunakan
- **Laravel:** Framework backend untuk penanganan data dan otentikasi yang aman.
- **MySQL:** Database relasional untuk menyimpan ribuan data aset secara efisien.
- **Bootstrap 5:** Antarmuka pengguna yang bersih dan responsif untuk dashboard admin.
- **DataTables:** Untuk pencarian, pengurutan, dan penyaringan daftar aset tingkat lanjut.
</Id>
